# StudyFAST - A Collaborative AI Learning Platform for FAST Students

StudyFAST is a web-based platform designed specifically for FAST students to share AI chat links related to academic topics. With students frequently relying on AI-generated explanations for complex concepts, this system provides a centralized platform to share and organize these resources for collaborative learning.

---

## 🚀 Project Objectives

- **Centralized Platform**: Create a hub for sharing AI-generated academic resources.
- **Structured Organization**: Enable the categorization of AI chat links by subject and tags.
- **Content Evaluation**: Allow students to evaluate content using upvotes, bookmarks, and comments.
- **Learning Analytics**: Provide personalized analytics and recommendations.
- **Collaboration**: Enhance knowledge sharing and peer validation among FAST students.

---

## 🛠️ System Modules

### 1. Authentication System

- Secure registration and login.
- User account management with role-based access (Student/Admin).

### 2. AI Chat Sharing

- Submit posts with Title, Subject/Course, AI Tool used, and Chat link.
- Include short descriptions and tags for better discoverability.
- Posts are displayed in a structured feed.

### 3. Search & Filter System

- Keyword-based search functionality.
- Filter by subject, tags, AI tool, and popularity.
- Sort by newest or most upvoted posts.

### 4. Community & Interaction

- **Voting**: Upvote/downvote system to reflect the usefulness of a post.
- **Bookmarks**: Save posts to a personal dashboard for later reference.
- **Comments**: Discussion section for clarification, correction, and peer validation of AI responses.

### 5. Learning Analytics

- Personal dashboard showing saved posts and interaction history.
- Insights into most viewed subjects.
- Basic recommendation system based on user activity.
